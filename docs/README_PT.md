# Sistema de Gestão de Uniformes

🌐 Language: Português | [English](README.md)

## Visão Geral
Sistema web para gerenciamento completo do fluxo:
pedidos → produção → estoque → entrega

## Funcionalidades
- Clientes
- Uniformes
- Pedidos
- Produção por etapas
- Fardos
- Entregas
- Estoque

## Fluxo
Cliente → Uniforme → Pedido → Itens → Produção → Fardo → Entrega → Estoque

## Vantagens
- Centralização total
- Rastreabilidade completa
- Redução de erros
- Controle por etapas
- Acesso remoto
