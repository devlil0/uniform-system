## Overview

Web system to manage the full workflow:

**Orders → Production → Inventory → Delivery**

Centralizes all company operations in one place, replacing spreadsheets and manual processes.

---

## Features

### Clients

- Client registration with name and contact information

### Uniforms

- Name
- Fabric: PV or BRIM
- Size: PP to EXG
- Color
- Reflective option

### Orders

- Create orders per client
- Order status:
  - PENDING
  - IN_PRODUCTION
  - COMPLETED
  - SHIPPED

### Order Items

- Link uniforms with quantities

### Production

- Assign seamstress
- Production stages:
  - CUTTING
  - SEWING
  - PRINTING
  - FINISHING
  - PACKAGING
- Track input and output per stage

### Bundles

- Group completed orders
- Track shipping date

### Delivery

- Assign delivery person
- Delivery status:
  - PENDING
  - IN_TRANSIT
  - DELIVERED

### Inventory

- Track available items
- Link items to specific clients

---

## System Flow

Client → Uniform → Order → Items → Production → Bundle → Delivery → Inventory
